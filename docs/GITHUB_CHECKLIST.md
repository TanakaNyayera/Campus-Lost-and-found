# GitHub Release Checklist

Use this before making the repository public.

- Confirm `config/config.yaml` is not committed.
- Confirm `.env` is not committed.
- Confirm runtime logs are not committed.
- Confirm chat uploads and proof uploads are not committed.
- Replace any real database or email credentials before sharing screenshots.
- Regenerate the Gmail app password that was previously stored locally.
- Add screenshots to the README if you want the repository to look stronger to recruiters.
- Run `python run_flask_local.py` locally and confirm the homepage loads.
